 Diagonal utama

↖️ Refleksi y = -x
[x'] [ 0 -1 ] [x] [y'] = [-1 0 ] [y]
 Diagonal sekunder
x' = -y (x menjadi -y)
y' = -x (y menjadi -x)
📏 Refleksi x = a
[x'] [2a-x] [y'] = [ y ]

y' = y (koordinat y tetap)
